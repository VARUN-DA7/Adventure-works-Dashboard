# Adventure-Works-Analytics
## Objective 
The goal was to create an interactive and insightful tool that enables Adventure Works Cycles company to monitor and enhance their sales strategies by identifying key trends and performance metrics across different regions, products, and customer segments.
## Tools Used 
 Ms EXCEL :
Excel was used for initial data cleaning, Data modelling using power pivot.
Pivot tables and charts in Excel were employed to create quick visualizations of year-wise sales amounts and other KPIs.

MYSQL :
SQL was used to query the relevant sales data for analysis.
Complex queries were written to filter, join, and aggregate data from multiple tables.
Data was cleaned and structured in SQL to ensure  it  was  ready  for  further  analysis  in other tools.

TABLEAU :
A data visualization tool used for creating advanced visualizations and interactive dashboards.
Tableau's ability to create interactive visualizations helped in exploring the data more comprehensively and identifying hidden patterns.

POWER BI :
Power BI was used to develop interactive reports and dashboards.
The tool’s capabilities in handling large datasets and creating dynamic visuals.
Power  BI's  integration  features  allowed  for combining data from various sources, enhancing the overall analysis.

## Challenges Faced
Converting raw and inconsistent data into Consistent data.

Extracting dates from inconsistent date key columns.

Merging and Data Modelling issues due to unstructured and nulls contained data.

Converting and Importing into SQL lead to loss of data.

Creating consistent KPI across all modules.

Modifying data format for the suitable requirement

## Overcoming Solutions 
Handled and Cleaned the data using ETL in Power Query.

Structured the date columns by changing the data types and using measures and calculated fields. 

Handled Nulls when merged with the three tables by replacing and removing nulls .

Cleaned the data in Power query then imported into SQL to avoid loss of rows/data.

Covered similar Insights and Charts across all modules.

## Insights/Suggestions & Recommendations 
Company should reduce their Production cost in order to increase their Operational Efficiency.

From the Analysis in every year the Production cost is higher than the Profits.

Company should also track the Sales that are moving up and down gradually. Campaigns /Product offers should run to see sales increase during off-peak seasons.

The most sold products are from bikes .They should also keep an eye on accessories and Gears.

 Most Valuable segment in Customer’s Occupation is ‘Professionals’ , need to be studied about another segments.

 ## Conclusions 
 The  project  has  successfully  uncovered  significant trends  and  patterns  in  the  sales portfolio, offering actionable insights that can be used to refine sales  strategies,  identify the sales trends,  and improve customer targeting. The use of tools like Excel, Tableau, Power BI, and SQL was critical in preparing comprehensive  dashboards  that  facilitate  informed decision-making.

Business Impact: These dashboards empower Adventure Works Cycles with actionable insights, enabling data-driven decisions that can enhance profitability, optimize product offerings, and strategically expand into underperforming regions.
